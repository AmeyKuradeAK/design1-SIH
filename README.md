# Project: E-commerce Website

## Description

This is an e-commerce website that allows users to browse and purchase products. The website is built using HTML, CSS, and JavaScript.

## Getting Started

To get started, clone the repository and install the dependencies.

```
git clone https://github.com/AmeyKuradeAK/design1-SIH.git
cd e-commerce-website
npm install
```

## Running the Website

To run the website, start the development server.

```
npm start
```

The website will be available at http://localhost:3000.

## Features

The website has the following features:

* Users can browse products by category.
* Users can view product details and add products to their cart.
* Users can checkout and purchase products.
* The website is responsive and mobile-friendly.

## Code Structure

The code is organized into the following directories:

* `src`: Contains the source code for the website.
* `public`: Contains the static assets for the website.
* `node_modules`: Contains the dependencies for the website.
